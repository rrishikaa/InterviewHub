
# InterviewHub

Interview Prep Hub is a logic-focused Next.js application built to practice and demonstrate core React and Next.js concepts commonly used in frontend interviews.

The project intentionally keeps styling minimal and focuses on state management, hooks, routing, and data flow.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript

## Features

- Mock authentication with protected routes
- Dashboard with aggregated stats
- Flashcards with keyboard navigation
- Timed MCQ quiz with score summary
- Notes system with full CRUD
- LocalStorage persistence
- API routes for quiz results and stats
- Loading and error states

## Pages

- `/login` – Mock login  
- `/dashboard` – Overview and stats  
- `/flashcard` – Flashcard practice  
- `/mcq` – Timed MCQ quiz  
- `/myNotes` – Personal notes (CRUD)
